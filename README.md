# Rank Tracker Website

A competitive gaming rank tracker displaying rankings across multiple platforms and game types.

**Live at:** https://senbosranks.github.io/

## Features
- 🎮 Track ranks across 50+ games
- 🎨 Dark theme with gradient UI
- 📊 Category filtering (FPS, Rhythm, MOBA, Card games, etc.)
- 📱 Fully responsive design
- ⚡ Fast and lightweight

## Games Tracked
- **Counter-Strike 2** - Premier & Faceit
- **Overwatch 2** - Competitive rankings
- **osu!** - Standard, Mania, Taiko, CTB modes
- **Valorant**
- And 40+ more games

## Customization
Edit the `games` array in `index.html` to update your rankings and add new games.
